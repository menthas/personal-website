Machine Learning or the interesting stuff

This has been my main area of interest in the past years. I've worked extensively on classification, regression (both linear and logistic) and ensemble methods. Check out my projects for the latest updates.
Some of my projects in recent years have focused on recommender systems and collaborative filtering mostly in in music recommendation. Currently I'm working on hybrid methods that add contextual data to increase the accuracy of CF.
I've recently been introduced to the field and am loving it so far. More projects on this front are coming.

Data Analysis or understanding and predicting

At the moment I'm focusing extensively on statistical data analysis of large datasets. To be specific this includes the challenges of model fits, statistical inference and model evaluation (parameter selection, regression, regularization, ANOVA's all over the place, ...). My favorite tool so far ? R !
The achieve all of the above one needs distributed computation! I'm proficient in Hadoop's MapReduce and Pig. Starting to work more on Spark and H2O (memory based frameworks) and of course coordinating all these mostly with Zookeeper.
What platform do I use? At home I either use local instances or Amazon EMR and the like. At work, secret architectures!
I worked with Bayesian methods in the context of machine learning but compared to frequentist methods, I still need to learn a lot about them.

Web or how it all started

Web is not just about programming and interfaces and I've mostly worked on the other half. That is, CDN's, complex caching using Varnish, Redis, etc. or load balancing using Apache Webserver or log aggregation and database management.
I started Python about 4 years ago and have loved it ever since. In the context of web I've worked with frameworks like Flask and Django but Python is my first tool for many things including Machine Learning, automation, testing theories and day to day hacking.
I started programming with PHP and worked with it for about 6 years before abandoning it because of many reasons described here. But nevertheless, I'm a certified Zend Engineer and I know my way around PHP, Symfony 2.0, Zend Framework, Yii and many other libraries and frameworks.
For many years I've worked with the presentation layer on the web including working with many JS/CSS frameworks. But more recently I'm working with Javascript/CSS outside of the browser. namely the V8 Engine and the Gnome 3 plugin development.
